# MyProject

A simple C++ project template.

## Building

### Prerequisites
- CMake 3.10 or higher
- C++17 compatible compiler (GCC, Clang, or MSVC)

### Build Instructions

```bash
# Create build directory
mkdir build
cd build

# Configure
cmake ..

# Build
cmake --build .

# Run
./MyProject
```

## Project Structure

```
.
├── CMakeLists.txt      # CMake build configuration

│   └── app.h
├── src/                # Source files
│   ├── main.cpp
```
